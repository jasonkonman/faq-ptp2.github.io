  <h1>דוגמא לתפריט יומי – יש להתאים את הכמויות באופן אישי:</h1>
  <h2>בוקר:</h2>
    <ul>
      <li>ירקות – חיים</li>
      <li>דגנים – לחם / לחמניה / פיתה קטנה (רצוי מקמח מלא או שיפון) או דייסה מדגן מלא (כגון שיבולת שועל) או דגני בוקר מלאים</li>
      <li>חלבון – גבינה רזה / יוגורט / טונה במים / ביצה / פסטרמה</li>
      <li>שומן – שמן זית / קנולה / סויה או טחינה או אבוקדו</li>
      <li>שתייה – ללא סוכר</li>
    </ul>
  <h2>ארוחת ביינים:</h2>
    <ul>
      <li>פרי - העונה</li>
      <li>שומן – אגוזים (מלך/פקאן) / שקדים / בוטנים / פיסטוקים לא מומלחים</li>
      <li>שתייה – ללא סוכר</li>
    </ul>
    <h2>צהריים:</h2>
    <ul>
      <li>ירקות – חיים ומבושלים</li>
      <li>דגנים ופחמימות – דגנים (רצוי מלאים) כגון אורז, בורגול, אטריות, קינואה, או קטניות כגון עדשים שעועית, אפונה, פול, או תפוח אדמה/ בטטה</li>
      <li>חלבון – בשר (עוף, הודו, דג, בקר דל שומן) ללא עור, איברים פנימיים ושומן נראה לעין, או תחליף צמחוני</li>
      <li>שומן – מעט שמן זית / קנולה / סויה להכנת הארוחה</li>
      <li>שתייה – ללא סוכר</li>
    </ul>
  <h2>ארוחת ביניים:</h2>
    <ul>
      <li>פרי – העונה</li>
      <li>שומן – אגוזים (מלך/פקאן) / שקדים / בוטנים / פיסטוקים לא מומלחים</li>
      <li>שתיה – ללא סוכר</li>
    </ul>
  <h2>ערב:</h2>
    <ul>
      <li>ירקות – חיים ו/או מבושלים</li>
      <li>דגנים – לחם / לחמניה / פיתה קטנה (רצוי מקמח מלא או שיפון)</li>
      <li>או דייסה מדגן מלא (כגון שיבולת שועל) או דגני בוקר מלאים</li>
      <li>חלבון – גבינה רזה / יוגורט / טונה במים / ביצה / פסטרמה</li>
      <li>שומן – שמן זית / קנולה / סויה או טחינה או אבוקדו</li>
      <li>שתיה – ללא סוכר</li>
    </ul>
  <h2>ארוחת לילה:</h2>
    <ul>
      <li>דגנים – לחם (רצוי מקמח מלא או שיפון)</li>
      <li>חלבון – גבינה רזה / יוגורט / טונה במים או שומן – טחינה או אבוקדו</li>
      <li>ירק – חי</li>
      <li>שתיה – ללא סוכר</li>
    </ul>
